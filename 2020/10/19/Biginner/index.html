Basic Computer Science for the Biginner

Despite I am not a real computer science student, but I believe everyone who is in this field shoudl know a bit of computer science.

What is Computer Science

Computer Science is the study of computers and computational systems. Unlike electrical and computer engineers, computer scientists deal mostly with software and software systems; this includes their theory, design, development, and application.

[University of Maryland] [https://undergrad.cs.umd.edu/what-computer-science]

There are many computer science disciplines out there, such as Applied Mathematics, Computer Engineering, Operating Systems, and Data Management, etc. How we can choose them it depends on yout interest.

In this series, I will be introduce some basic computer science knowledge for the biginner. You must have a solid foundation if you’re going to have a strong superstructure. Another reason why we should learn some CS knowledge is that many tech companies will ask you some related questions during the interview. Having a broad knowledge can help you to stand out. We may don’t have to master all of them; at least we have to know about them.

Let’s get start

Data

Data is one of the most important areas that programmers have to understand. Knowing how data works and how to store them are basic knowledge for us. I will list some knowledge that I think is importnat.

Check List

  • Main Memory
    1). Memory is the place where data is stored. A computer contains a large collection of circuits. Each capable of storing a single bit. This bit reservoir is known as the main memory (AKA RAM, random access memory).
    2). A computer’s main memory is organized in manageable units called cells. A cell contains 8 bits.
    3). Each cell is assigned a unique name is called address. All the cells being placed in a single row and numbered in this order starting with 0.

  • Information is encoded as pattern of 0s and 1s. These digits are callede bits. Hence 1 cell = 8 bits = 1 byte
    1). Because 0 == false, 1 == true; therefore, it leads to next topic Boolean Operations
    2). Boolean Operations is a way to store and manipulate inside a computer
    3). Four basic Boolean Operations are : AND, OR, XOR, NOT

  • Gates and Filp-Flops
    1). A device that produces the output of a boolean operation when given the operation’s input values is called a gate.
    2). Filp-flop is a circuit that produces an output value of 0 or 1, which remains constant until a pulse ( a temporary change to a 1 that returns to 0 ) from another circuit causes it to shift to the other value.

    To illustrate the relationship between Boolean Operations and circuit we can see a illustration below:

    The output will flip or flop between two values under control of seternal stimuli

  • Hexadecimal Notation
    1). The internal activities of a computer is conducted by patterns of bits. (aka a string of bits). Stream if the string has a long length.
    2). However, Stream is difficut to read; its pattern is like 101010100101011. So we use hexadecimal. 

    | Bit Pattern | Hexadecimal |
| :---        |         ---:|
| 0001        |      1      |
| 1010        |      A      |

    hexadecimal notation uses a single symbol to represent a pattern of 4 bits

  • Measuring Memory Capacity
    1). KB - kilobyte, MB - megabyte, GB - gigabyte, TB - terabyte
    2). Most computers have additional memory devices called mass sotrage or secondary storage. Such as DVD, USB, Flash Drives, etc.

  • Representing Information as Bis Patterns
    1). Ws used to use ASCII, American Standard Code for Information Interchange. But it can only use for english. Now, we use Unicode, it is more powerful and it consists of 65,536 different bit patterns- enough to allow text written in such languages as Chinese and some East European languages.
    2). Unicode uses a pattern of 16 bits to represent each symbol. In contrast, ASCII uses a pattern of 8 bits.
